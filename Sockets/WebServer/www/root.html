You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- use to multiply two integer values together.
    • for example, "18.117.243.97/multiply?num1=6&num2=4"
  • /github -- use to list out all of a user's repos and their associated data.
    • for example: "18.117.243.97/github?query=users/amehlhase316/repos"
  • /comment -- use to add comments to the list below!
    • for example: "18.117.243.97/comment?text=despite all my rage i am just a rat in a cage&author=trenton mosher"
  • /printpw -- use to print out the powers of a number in sequence up to a defined limit
    • for example: "18.117.243.97/printpw?num1=2&num2=5"
  • /comments.txt -- expands the comments file for easier reading!
    • for example: "18.117.243.97/comments.txt"
