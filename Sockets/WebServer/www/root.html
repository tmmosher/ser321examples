You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- use to multiply two integer values together.
    • for example, "/multiply?num1=6&num2=4"
  • /github -- use to list out all of a user's repos and their associated data.
    • for example: "/github?query=users/amehlhase316/repos"
  • /comment -- use to add comments to the list below!
    • for example: /comment?comment?text=despite all my rage i am just a rat in a cage&author=trenton mosher
